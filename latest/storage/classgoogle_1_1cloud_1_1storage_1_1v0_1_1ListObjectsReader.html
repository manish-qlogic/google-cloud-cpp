ListObjectsReader::iterator google::cloud::storage::v0::ListObjectsReader::begin ( )

Return an iterator over the list of objects.

The returned iterator is a single-pass input iterator that reads object metadata from the BucketListReader when incremented.

Creating, and particularly incrementing, multiple iterators on the same RowReader is unsupported and can produce incorrect results.

Retry and backoff policies are honored.

Exceptions
std::runtime_errorif the read failed after retries.
