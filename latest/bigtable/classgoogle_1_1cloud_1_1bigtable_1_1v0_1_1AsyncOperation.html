Detailed Description

Represents a pending asynchronous operation.

When applications create an asynchronous operations with a CompletionQueue they provide a callback to be invoked when the operation completes (successfully or not). The completion queue type-erases the callback and hides it in a class derived from AsyncOperation. A shared pointer to the AsyncOperation is returned by the completion queue so library developers can cancel the operation if needed.
